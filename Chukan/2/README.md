# 図書館業務管理システム

## 概要

Javaで作成したコンソール版の図書館業務管理システムです。

書籍の登録・検索・貸出・返却・ジャンル管理などを行うことができます。

本システムはJavaの基本文法、オブジェクト指向、コレクション、ラムダ式を学習することを目的として開発しています。

## 主な仕様

### ISBN

書籍登録時に

978 + ランダム10桁

を自動生成します。

重複チェックを行い、一意の番号を採番します。

---

### ジャンル

- 最大3件
- 重複登録不可

---

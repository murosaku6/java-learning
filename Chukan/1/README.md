# タスク管理システム

## 概要

Javaで作成したコンソール版のタスク（ToDo）管理システムです。
タスクの登録・一覧表示・検索・更新・削除・ソートなどを直感的に操作することができます。

本システムは、Javaの基本文法、オブジェクト指向、コレクション、ラムダ式を学習・活用することを目的として開発されています。

---

## 開発環境

- Java 25
- Visual Studio Code
- GitHub Codespaces
- Git / GitHub

---

## 使用技術・要素技術

- **オブジェクト指向**: 役割に応じたクラス設計（モデル、ロジック、プレゼンテーション）
- **コレクション**: `ArrayList` を用いた動的なタスクデータ管理
- **関数型プログラミング**: ラムダ式および `Stream API` による効率的なデータ処理・フィルタリング
- **ソート**: `Comparator` による動的な並び替え
- **例外処理**: ユーザーからの予期しない入力（型不一致など）に対する堅牢なバリデーション

## 機能一覧

### 1. タスク管理 (CRUD)
- **タスク登録**: タイトル、期限、優先度（高/中/低）を指定して登録。IDは自動で連番採番されます。
- **タスク一覧表示**: 登録されたタスクの情報を一覧形式で画面に出力。
- **タスク更新**: タスクのステータス（未着手 / 進行中 / 完了）や詳細を変更。
- **タスク削除**: 不要になったタスクをID指定で削除。

### 2. 高度な検索・フィルタリング
- **ID検索**: タスクIDでのピンポイント抽出。
- **キーワード検索**: タイトルの一部からあいまい検索。
- **ステータス別検索**: 特定の進捗状況（例：「完了」のみ）のタスクを絞り込み。
- **優先度別検索**: 優先度「高」のタスクのみをフィルタリング。

### 3. ソート・並べ替え
表示順を以下のオプションで動的に変更できます：
- 登録順（ID昇順）
- 期限順（締切が近いものから表示）
- 優先度順（「高」を最優先で表示）

## クラス構成（アーキテクチャ）

```text
[Main.java] （プレゼンテーション層 - CLI入力・メニュー表示）
   │
   ▼ （ユーザーの指示を伝達）
[TaskManager.java] （ビジネスロジック層 - リスト操作・検索・ソート処理）
   │
   ▼ （データを保持）
[Task.java] （モデル層 - 個々のタスクオブジェクト）
```

## 学習のポイント

- **カプセル化と責務の分離**: 画面表示（`Main`）、タスク管理（`TaskManager`）、データモデル（`Task`）に適切に分割することによる、変化に強いコード設計の理解。
- **Stream APIの実践**: `filter` や `collect` を用いた簡潔かつ高速なデータ絞り込みロジックの習得。
- **動的ソートの実装**: `Comparator.comparing` などを使用した、状況に合わせた並べ替えの実装方法の理解。
- **例外ハンドリングの実践**: コンソールアプリでの入力エラーを捕捉し、アプリが強制終了しない仕組みづくりの学習。

# タスク管理システム

## 概要

Javaで作成したコンソール版のタスク管理システムです。

タスクの登録・一覧表示・検索・削除・完了状態の切り替えなどを行うことができます。

Javaの基本文法、オブジェクト指向、コレクション、ラムダ式、例外処理を学習することを目的として開発しました。

---

## 機能一覧

### タスク管理

- タスク登録
- タスク一覧表示
- タスク検索
- タスク削除
- タスク完了・未完了切替

### 一覧表示

- 登録順
- タイトル順（昇順）
- タイトル順（降順）
- 完了状態順

---

## 主な仕様

### タスクID

タスク登録時に1から順番に自動採番します。

### 検索

- ID検索
- タイトル検索

### 入力チェック

- 数字入力チェック
- 存在しないIDのチェック
